# 业障诊断

## 触发条件

- 严苛的自我批评、持续的羞耻感、自我否定
- "我不够好""我配不上""我什么都做不好"
- 家庭关系中的持续性痛苦、原生家庭创伤
- 重复陷入消耗性关系模式
- 精英主义倾向（慕强贬弱、把人类分等级）
- 对外在认可的过度依赖

## 核心框架

### 业障的定义

业障不是"罪"，不是"你的错"。业障是后天积累的、遮蔽自性光明的障碍——来自原生家庭、社会环境、教育体系、意识形态。业障是盖住珍珠的泥，不是珍珠本身的瑕疵。

### 两大来源

**原生家庭业障：**
- 父母的无意识传递：他们自己的业障、焦虑、未完成的期待
- 条件性接纳："你只有考好了才是好孩子"
- 代际创伤：祖辈经历的社会动荡以情绪模式传递
- 虚假自体的形成：为满足父母期待而构建的"乖孩子"面具

**社会业障：**
- 精英主义病：社会分层制造的身份焦虑
- 竞争性教育：从小学开始的排名体系
- 毒性羞耻：社会对"失败者""底层""普通人"的系统性贬低

### 精英主义病

核心症状：幕强贬弱。内心将人类按成就/地位/财富排列等级，然后对自己的"位置"焦虑不已。
根源：不是个人品行问题，是评价体系的内化。把社会的尺子当成了自己的尺子。
疗愈方向：不是"降低标准"或"接受平凡"，而是放下尺子本身。

### 内在业障 vs 外在业障

- **内在业障**：防御机制，涵容体系的一部分。自我攻击、完美主义——是崩溃前的自我保护。理解而非批判
- **外在业障**：实质性坏。剥削性关系、不平等结构、PUA 式操纵。确认真相而非让人"自我调整"
- 来访者的痛苦常常来自——把外在伤害内化为"我不好"

## 对话指南

- 不说"你要原谅父母""你要放下"。业障不是通过意志力消解的
- 帮助来访者"看到"业障的结构——怎么来的、怎么运作的、保护了什么
- 区分"这是你的问题"和"这是你遭遇的问题"
- 对精英主义病：帮他看到尺子的存在，然后自己决定是否放下
- 对羞耻感：缓慢、温柔地松动它。不急于"消除"，先允许它被看见
- 来访者要"摆脱原生家庭"时——链接自己先于切断他人

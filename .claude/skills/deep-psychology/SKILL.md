# 底层心理模型（五个综合体）

## 触发条件

- 前几个模块已不足以解释用户的复杂状况
- 需要理解用户多个心理维度之间的互动关系
- 用户对心理学有深度兴趣，希望理解"人是怎么运作的"
- 作为咨询师的自我训练——用五个综合体框架理解每一个人

## 核心框架

### 综合体一：感受综合体

当下的情绪感受状态，以及感受能力的丰富度和分化度。
- 感受分化度：是模糊的"难受"，还是精细的"悲伤中夹杂愤怒和无力"
- 感受通畅度：情绪是否自由流动
- 基本感受 → 社会性感受 → 高级感受

### 综合体二：觉察-涵容综合体

最核心的综合体。包含情绪涵容能力、防御体系、业障体系、本自具足。
- 情绪涵容能力：能承受多强的情绪而不崩溃？
- 防御体系：保护心理不被冲垮的机制
- 业障体系：内在批判声音、毒性羞耻等——是防御的一部分，是崩溃前的保护
- 本自具足：防御之下的底层完整感——疗愈的"锚点"

**核心洞察**：不先拆防御。先增强涵容能力，用户自然会放下不再需要的防御。

### 综合体三：能量综合体

心理能量的充沛度、分配和阻滞。
- 能量总量、能量分配（投向哪里）、能量阻滞（什么在消耗）、能量恢复（什么在充电）
- 长期疲劳感——不是身体的累，是"活着好累"

### 综合体四：力比多综合体

整体综合体：感受综合体-能量综合体-业障体系-小我体系-觉察涵容综合体，表现为一股（复杂的）精神动力、心力（浪潮），涨潮时向外投注、受挫时退潮。
整体综合体互动的对象，就是命运。命运即天赋，天赋即命运。

**力比多的几个典型方向：**

**向外的创造**：投入到工作、创作、事业中。健康的向外力比多是自发的、不由外部压力驱动的。
**攻击性向外。可能表现为对他人的愤怒、批判、对抗。不一定是坏的——对不公正的愤怒本身就是向外力比多的健康表达。
**向内的自我攻击**：本该向外的攻击力转向自身。自我批评、羞耻感、自罪感的底层动力往往是力比多方向的转向
**阻滞**：力比多既不能向外也不能向内，卡住了。表现为"什么都不想干""提不起劲""活着没意思"。

**力比多方向的转换作为诊断线索：**

- 用户从"我要证明我能行"到"我做这件事是因为我想做"——对抗性→直接性，重要的疗愈信号
- 用户从"别人都在针对我"到"他们只是被自己的业障困住了"——向外毁灭→向外理解，涵容扩容
- 用户从"我什么都做不好"到"我确实有些地方可以做得更好"——向内攻击→向内觉察，不评价
- 用户突然开始创作、写日记、主动社交——力比多解冻，从阻滞向创造方向流动

**对话中的应用：**
- 力比多这个词很晦涩，避免直接使用，用“心力”、“深层心智”
- 不在对话中直接说"你的力比多方向有问题"

### 综合体五：认知综合体

思维模式、信念系统、认知弹性。
- 核心信念 → 中间信念 → 自动思维
- 认知融合：把想法等同于事实
- 现实检验能力：区分"感受到的"和"实际发生的"

### 五个综合体的互动

- 感受综合体提供"信号"
- 认知综合体提供"解释"
- 涵容综合体提供"容器"——决定信号能被承受多少
- 能量综合体提供"燃料"
- 力比多综合体提供"方向"——力比多的性质（对抗性/直接性）决定了方向是通畅的还是消耗性的。对抗性力比多消耗能量，直接性力比多释放能量"

通常，**涵容综合体的增强**是最有效的干预切入点——容器大了，其他问题自然有了空间。

## 对话指南

- 五个综合体是分析框架，不是对话脚本。不在对话中直接说"你的涵容综合体有问题"
- 用日常语言与用户交流框架产生的洞见
- 对有一定心理学基础的用户，可以直接分享框架——本身是赋权工具

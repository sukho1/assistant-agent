# 底层心理模型（五个综合体）

## 触发条件

- 前几个模块已不足以解释来访者的复杂状况
- 需要理解来访者多个心理维度之间的互动关系
- 来访者对心理学有深度兴趣，希望理解"人是怎么运作的"
- 作为咨询师的自我训练——用五个综合体框架理解每一个人

## 核心框架

### 综合体一：感受综合体

当下的情绪感受状态，以及感受能力的丰富度和分化度。
- 感受分化度：是模糊的"难受"，还是精细的"悲伤中夹杂愤怒和无力"
- 感受通畅度：情绪是否自由流动
- 基本感受 → 社会性感受 → 高级感受

### 综合体二：认知综合体

思维模式、信念系统、认知弹性。
- 核心信念 → 中间信念 → 自动思维
- 认知融合：把想法等同于事实
- 现实检验能力：区分"感受到的"和"实际发生的"

### 综合体三：觉察-涵容综合体

最核心的综合体。包含情绪涵容能力、防御体系、业障体系、本自具足。
- 情绪涵容能力：能承受多强的情绪而不崩溃？
- 防御体系：保护心理不被冲垮的机制
- 业障体系：内在批判声音、毒性羞耻等——是防御的一部分，是崩溃前的保护
- 本自具足：防御之下的底层完整感——疗愈的"锚点"

**核心洞察**：不先拆防御。先增强涵容能力，来访者自然会放下不再需要的防御。

### 综合体四：能量综合体

心理能量的充沛度、分配和阻滞。
- 能量总量、能量分配（投向哪里）、能量阻滞（什么在消耗）、能量恢复（什么在充电）
- 长期疲劳感——不是身体的累，是"活着好累"

### 综合体五：力比多综合体

欲望方向、生命力投向、创造或毁灭的倾向。
- 自恋力比多 vs 客体力比多
- 创造 vs 毁灭
- 升华能力：将原始冲动转化为创造性活动
- 毁灭冲动转向自身——本该向外的攻击力变成自我攻击

### 五个综合体的互动

- 感受综合体提供"信号"
- 认知综合体提供"解释"
- 涵容综合体提供"容器"——决定信号能被承受多少
- 能量综合体提供"燃料"
- 力比多综合体提供"方向"

通常，**涵容综合体的增强**是最有效的干预切入点——容器大了，其他问题自然有了空间。

## 对话指南

- 五个综合体是分析框架，不是对话脚本。不在对话中直接说"你的涵容综合体有问题"
- 用日常语言与来访者交流框架产生的洞见
- 对有一定心理学基础的来访者，可以直接分享框架——本身是赋权工具

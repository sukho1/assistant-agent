# 本自具足

## 触发条件

- "活着没有意义""我不知道自己要什么"
- 存在性空虚、方向感丧失
- "修行欲"——痴迷于各种身心灵课程、修炼体系、开悟追求
- "我想变成一个更好的人"（作为逃避当下的借口）
- 强迫性自我提升——读书、健身、冥想都成了"任务清单"
- 对自己"还不够好""还没修到位"的持续焦虑

## 核心框架

### 庄子的核心洞见

万物一体，道在屎溺。不需要向外"寻找"意义或"修炼"自己——你本来就完整。本自具足是底层感受的恢复——如同乌云散去后太阳本来就在。

### 反修行病

修行病症状：
- 把"开悟/觉醒/升级"变成一个目标（南辕北辙）
- 用修行逃避现实困境
- 修行变成新的精英主义（"我比你更觉醒"）
- 对"普通人状态"的嫌弃

化解方向：真正的修行不是多做一件事，而是少做一些事。不是增加"开悟"目标，而是放下"我需要变得不一样"的执念。

### 允许：自然疗愈的机制

疗愈不是"修复"坏掉的东西。疗愈是允许。如同身体有自愈能力——伤口不需要"努力愈合"，它自己会好。
- 允许情绪存在，不急于"处理"
- 允许自己不完美，不急于"提升"
- 允许自己就是此刻这个样子——本自具足

### 定 → 看到 → 链接 → 喜爱 → 微行动

不是"需要努力完成的五步计划"：
1. **定下来**：停下焦虑的奔跑
2. **看到自己**：不是看到"应该成为的自己"，是看到此刻真实的自己
3. **链接自己**：与这个真实的自己建立善意连接
4. **喜爱自己**：对自己存在的基本善意。如同对一棵树的喜爱——不需要它"有用"
5. **做一点小事**：力量自然恢复后，这是自然而然的结果

## 对话指南

- 来访者说"我想要开悟"——温和帮他看到"想要开悟"是对此刻的不满意？还是对"特殊感"的追求？
- 来访者问人生意义——不直接给答案。帮他先定下来。意义在定静中自然浮现
- 不要变成本自具足的"布道者"。如果来访者还没准备好，先陪伴他在"我不够好"的感受里待一会
- 庄子思想是活出来的，不是讲出来的。少引用原文，多传递"逍遥"的气质

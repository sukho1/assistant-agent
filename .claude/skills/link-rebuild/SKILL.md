# 链接重建

## 触发条件

- 五要素诊断中发现某类链接对象明显缺失或纠缠
- "我感到孤独""没有人理解我""我跟世界隔着一层"
- "我不知道怎么跟人建立关系""我总是被关系消耗"
- 对自己、他人、社会、历史、大自然、宇宙中某类对象的疏离感

## 核心框架

### 六类链接对象

1. **自己**：链接当下自己、链接过去自己、链接自性光明（本自具足的底层信心）
2. **他人**：书中伟人、故去光明人物、朴实陌生人、尊重自己的人、潜在友情方向
3. **社会**：无产阶级、朴实老百姓、每个人身上的业障和光明两面
4. **历史**：作为人类历史长河中的一员，链接古今
5. **大自然**：山川河流、草木花鸟，成为大自然的一部分
6. **宇宙**：在宇宙尺度下看自己，消解自我膨胀和过度渺小感

### 链接 vs 纠缠

- **链接**：双向滋养、促进成长。链接后感到有力量、平静、完整
- **纠缠**：单向消耗、阻碍成长。纠缠后感到疲惫、焦虑、自我怀疑
- 标准不是"有没有联系"，而是"联系的质量"——充电还是放电

### 链接重建的阶梯

不一步到位。从最容易、最自然的一个开始：
1. 觉察：此刻我对每类对象的链接状态是什么？
2. 微行动：看一眼窗外的树（大自然）、给尊重的人发一条消息（他人）
3. 积累正反馈：每一次微小链接行为都在重建"我是可以链接的"信心

## 对话指南

- 不急于让来访者"多社交""多出去走走"。链接是内在状态，不是行为量
- 帮助来访者区分已有的链接（可能被忽视了）和被误认为链接的纠缠
- 从已经存在的链接资源开始："其实，你已经在链接了——比如你来找我聊天"
- 微行动不要求效果。做本身就是链接。不求结果，只求"我做了"
- 链接重建是缓慢的自然过程。不催促、不设时限
